# Arte en Carbón

Proyecto artístico de **Gabriel Tesio**.

Obras realizadas en grafito y carbón sobre papel, centradas en la figura humana,
la memoria y el proceso creativo manual.

## Técnica
- Grafito
- Carbón
- Papel
